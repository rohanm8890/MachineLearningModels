# MachineLearningModels

Machine Learning Models for NR Link Adaptation / MCS Prediction

This project focuses on applying machine learning models to a 5G NR link adaptation dataset in order to predict the optimal Modulation and Coding Scheme, or bestMCS, using radio/network features such as CQI, RI, PMI, and optional channel-quality features when available. The models were built to evaluate how different ML approaches perform on the same prediction task, using consistent preprocessing, train/test splitting, and evaluation metrics such as exact accuracy, within ±1 MCS accuracy, over-prediction rate, and under-prediction rate.
XGBoost Model: Built an optimized XGBoost multi-class classification model to predict the best MCS value for 5G NR link adaptation using CQI, RI, PMI, and optional channel-quality features. The model remaps MCS labels into valid class indices, uses a train/validation/test split, and tests multiple configurations with early stopping to improve performance and prevent overfitting. It evaluates results using exact accuracy, within ±1 MCS accuracy, over-prediction rate, under-prediction rate, and confusion matrix analysis, making it useful for comparing both strict prediction accuracy and practical link-adaptation reliability.

Deep Neural Network Model: Developed a PyTorch-based multilayer perceptron to predict best MCS values from CQI/RI/PMI-based wireless features. The model uses standardized inputs, two hidden layers with ReLU activation, dropout regularization, class-weighted cross-entropy loss, and early stopping based on validation loss. This model demonstrates experience with deep learning workflows, including tensor preparation, dataloaders, model training, validation tracking, GPU/CPU device handling, and classification evaluation for a real wireless communication dataset.

ExtraTrees Model: Implemented an ExtraTrees ensemble classifier as a strong tree-based baseline for MCS prediction. The model trains 1,200 randomized decision trees using CQI, RI, PMI, and optional channel features, while applying balanced class weights to handle uneven MCS label distribution. It captures nonlinear relationships in the dataset without requiring heavy manual tuning and is evaluated using exact accuracy, within ±1 MCS accuracy, over-prediction rate, under-prediction rate, and confusion matrix shape for consistent comparison against the other models.

Linear SVM Model: Created a Linear Support Vector Machine model using a Scikit-learn pipeline with feature standardization and balanced class weights. The model predicts best MCS values from the same CQI/RI/PMI-based feature set and serves as a simpler linear baseline compared to the more complex tree-based and neural network models. Its purpose is to test how well a linear decision boundary can approximate the relationship between wireless link-quality indicators and optimal MCS selection, while still using the same evaluation metrics for fair benchmarking.
